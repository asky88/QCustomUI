Member Function Documentation

[static] std::optional<QByteArray> QCtmFile::load(const QString &file, QIODeviceBase::OpenMode mode = QIODevice::ReadOnly)

加载文件 file, 可设置文件打开模式 mode, 如果读取文件失败返回 std::nullopt，读取成功返回文件内容.

See also save.

[static] bool QCtmFile::save(const QString &file, const QByteArray &content, QIODeviceBase::OpenMode mode = QIODevice::WriteOnly)

保存内容 content 到文件 file 中，可设置文件打开模式 mode, 如果保存成功则返回true，否则返回false.

See also load.
